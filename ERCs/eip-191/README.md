# ERC2135 RefImpl

## Getting Started

```sh
yarn install
yarn test
```

When ready to deploy, fill in credentials in [The `.env.tmpl` file](./.env.tmpl) and change name to .env

```sh
yarn start # to deploy to goerli
```
